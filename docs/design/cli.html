CLI Design

The CLI demonstrates the standard command layout: a thin entry point, one file per public command, command-level help, and version reporting.

check

Runs basic repository conformance checks for required files, docs, Rack, design-doc status markers, lockfile, secret hygiene, and Hatchling build backend. A local root .env is allowed only when Git reports it as untracked and ignored. C/C++ profiles must also expose a Lizard-based native complexity gate under tests/.

standard

Prints a standard profile summary as text or JSON. The default profile is python-package; --profile cpp-library renders the C++ native profile, --profile csharp-app renders the C# profile, --profile javascript-web-app renders the no-build browser JavaScript/CSS profile, and --profile python-js-app renders the Python-served browser app profile.

version

Reports the package version, Python version, and major Wavenumber dependency versions.
