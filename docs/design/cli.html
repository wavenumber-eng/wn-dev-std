CLI Design

The CLI demonstrates the standard command layout: a thin entry point, one file per public command, command-level help, and version reporting.

check

Runs basic repository conformance checks for required files, docs, Rack, lockfile, secret hygiene, and Hatchling build backend.

standard

Prints a standard profile summary as text or JSON. The default profile is python-package; --profile cpp-library renders the C++ native profile, and --profile python-native-wasm renders the mixed-mode native/WASM profile.

version

Reports the package version, Python version, and major Wavenumber dependency versions.
