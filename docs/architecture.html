Architecture

wn-dev-std is a small installable Python package and the reference example for Wavenumber standards.

Boundaries

  • wn_dev_std.cli.main owns parser construction and dispatch.
  • Each public command owns its implementation in a separate command file.
  • wn_dev_std.standards owns public standard data examples.
  • wn_dev_std.checks owns reusable conformance checks.

Future Profiles

The base rules will be extended with C, C++, C#, JavaScript, Rust, native binary, WASM, and Zephyr profiles. Zephyr profiles must respect Zephyr layout and test tools such as west, twister, and ztest.
