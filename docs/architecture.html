Architecture

wn-dev-std is a small installable Python package and the reference example for Wavenumber standards.

Boundaries

  • wn_dev_std.cli.main owns parser construction and dispatch.
  • Each public command owns its implementation in a separate command file.
  • wn_dev_std.standards owns public standard data examples.
  • wn_dev_std.checks owns reusable conformance checks.

Future Profiles

The first extension profiles are cpp-library and python-native-wasm. The C++ profile captures the native CMake, Ninja, clang-format, clang-tidy, CTest, and ABI rules. The mixed-mode profile builds on that for Geometer-style packages that combine Python, CMake/C++, platform wheels, and WASM outputs. Future profiles will add C, C#, JavaScript, Rust, and Zephyr. Zephyr profiles must respect Zephyr layout and test tools such as west, twister, and ztest.
