# 2026-07-18

Release version: `2026.7.18`

## Changes

- Adds release-mode artifact governance for promoted payload catalogs,
  bounded distribution globs, declared version/source/sha metadata, and
  release-channel signoff.
- Adds TypeScript-first standards for browser and Python-served TypeScript
  projects, including strict compiler guardrails, JSONC config parsing,
  local-file `tsconfig` inheritance, package script checks, and migration
  metadata for JavaScript ports.
- Adds Rust standards for host applications/libraries and embedded firmware,
  including Cargo metadata checks, root lockfile policy, rustup toolchain
  checks, per-package unsafe lint governance, workspace inheritance,
  `workspace.exclude`, firmware target/runner metadata, and explicit Cargo
  test/doctest/rustdoc lanes.
- Updates public CLI/config/interface contracts, audited design docs, README
  guidance, and release governance examples for the combined standard update.

## Validation

- External review completed for the release-mode, TypeScript, and Rust slices.
- `uv lock --check` passed after updating the root package lock metadata.
- `uv run rack run --all` passed locally with 359 passed, 1 skipped, and all
  27 Rack subtests passing.
- `uv run python -m build` produced `2026.7.18` wheel and sdist artifacts.
- `uv run twine check dist/*` passed.
- `uv run dev-std audit . --scope docs.release --mode release` passed against
  the release catalog.
