# Release Notes: 2026-07-01

## `2026.7.1`

Zephyr firmware now uses the upstream C++ clang-format baseline.

- Removes the separate Zephyr Attach/right-pointer/no-sort formatter policy.
- Makes the Zephyr profile audit use the same C++ clang-format settings as
  `cpp-library`.
- Updates the reusable Zephyr `.clang-format` template to Allman braces, left
  pointer alignment, sorted includes, and preserved include blocks.
- Adds regression coverage so Zephyr profile projects continue to inherit the
  C++ formatter baseline.
