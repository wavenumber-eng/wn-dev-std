# Release Notes: 2026-06-04

## `2026.6.4`

Initial public Python standards package.

- Adds installable `wn-dev-std` package and CLI.
- Adds strict Python package baseline documentation.
- Adds the first C++ baseline for CMake/Ninja projects with clang-format,
  clang-tidy, CTest, warnings, sanitizer, ABI, and header-boundary rules.
- Adds the first mixed-mode baseline for Geometer-style Python/native/WASM
  packages.
- Adds Rack-based L0 and L99 signoff.
- Adds CI and release-validation workflow examples for Ubuntu, Windows, macOS,
  Hatchling, optional PyPI trusted publishing, and GitHub Release-published
  releases.
- Initial releases kept `wn-dev-std` model-only; PyPI publication begins with
  the 2026.6.28 release.
