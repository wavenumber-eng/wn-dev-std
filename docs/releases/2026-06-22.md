# Release Notes: 2026-06-22

## `2026.6.22`

Public PR hygiene standard update.

- Adds a reusable `.github/workflows/pr-hygiene.yml` template that requires a
  same-repo linked issue, Conventional Commit PR titles, Conventional Commit
  commit subjects, 72-character commit subjects, no `WIP` subjects, no emoji,
  and no Claude/Anthropic/generated-with attribution.
- Adds a matching `.github/pull_request_template.md` template with
  `Linked issue:`, summary, validation, and Conventional Commit title guidance.
- Adds opt-in `[pr_hygiene]` validation for repos that have installed the
  workflow and PR template.
- Updates all standard profiles to include the public PR hygiene rule.
