# Release Notes: 2026-06-28

## `2026.6.28`

Plan exit-criteria and PyPI publication update.

- Adds structured plan exit criteria as a first-class plan metadata surface.
- Requires active/pending/blocked plans to declare `[[exit_criteria]]`.
- Reports exit-criteria status from `dev-std plan list` and `dev-std plan show`.
- Makes newly created plans include a default pending signoff exit criterion.
- Documents the exit-criteria lifecycle in the audit, CLI, and documentation
  standards.
- Configures `wn-dev-std` as a PyPI-published package.
