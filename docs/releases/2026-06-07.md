# Release Notes: 2026-06-07

## `2026.6.7`

Documentation status marker standard.

- Adds the documentation standard for HTML design docs.
- Requires `data-doc-status` markers with `draft`, `proposal`, `accepted`, or
  `superseded` values on HTML files under `docs/design`.
- Updates `wn-dev-std check` to fail unmarked or invalid design docs.
- Reports draft/proposal design docs in the check output so release signoff can
  review unfinished design surfaces intentionally.
- Marks the reference `wn-dev-std` design docs as accepted.
