# Release Notes: 2026-06-10

## `2026.6.10`

Secret hygiene and native complexity standard update.

- Allows a local root `.env` to pass `wn-dev-std check` only when Git reports
  the file as untracked and ignored.
- Continues to fail tracked `.env` files and root `.env` files whose ignore
  status cannot be verified.
- Adds an explicit Lizard-based native complexity rule to C/C++ and mixed-mode
  standards.
- Requires C/C++ profiles to expose a Lizard complexity gate under `tests/`.
