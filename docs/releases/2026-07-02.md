# 2026-07-02

Release version: `2026.7.2`

## Changes

- Adds governance audit scopes for ADRs, requirements, domains, governed
  surfaces, traceability refs, generated governance links, artifacts, vendors,
  release channels, and canonical build docs.
- Adds `docs.build`, requiring `docs/build.html` or `docs/build.md` with
  machine-readable metadata and minimum coverage of setup/tools,
  commands/invocation, outputs/artifacts, and validation/signoff.
- Adds tracked-file-first artifact and vendor audits using authored catalogs
  under `docs/governance/`.
- Adds release-channel governance requiring both `process_doc` and `build_doc`.
- Adds generated governance HTML support and governance link resolution.
- Adds ADR and requirement create/list/show command surfaces.
- Tightens plan/log governance with structured steps, exit criteria, and
  step-linked logs.

## Follow-Up
