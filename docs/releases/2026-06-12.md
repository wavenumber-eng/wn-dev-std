# Release Notes: 2026-06-12

## `2026.6.12`

Zephyr firmware and native signoff standard update.

- Adds the `zephyr-firmware` profile for west-based firmware projects.
- Adds reusable C/C++ and Zephyr signoff templates under `docs/templates/`.
- Sets the canonical new-code native complexity threshold to
  `max_cyclomatic_complexity = 10`.
- Adds checks for native signoff configuration, including file size, function
  size, complexity, clang-format mode, clang-tidy mode, and Lizard enforcement.
- Documents Rack as the validation model for all projects and points to the
  public `wn-rack` package on PyPI.
- Documents that every project needs an `L99_signoff` gate with measurable
  checks such as complexity, file size, function size, formatting, static
  analysis, docs, contracts, release metadata, and project-local ratchets.
