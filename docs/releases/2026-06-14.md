# Release Notes: 2026-06-14

## `2026.6.14`

JavaScript documentation-layout standard update.

- Accepts the foldered JavaScript standard design doc path
  `docs/design/standards/javascript.html` for `javascript-web-app` and
  `python-js-app` projects.
- Keeps the legacy `docs/design/javascript-standard.html` path valid for
  existing repositories.
- Adds `[documentation.standard_docs].javascript` so projects can declare a
  deliberate custom canonical JavaScript standard design-doc path.
- Updates the configuration schema, JavaScript standard design doc, README, and
  regression tests for the accepted and configured paths.
