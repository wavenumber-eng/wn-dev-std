# Release Notes: 2026-06-29

## `2026.6.29`

Plan hygiene migration ignore support.

- Adds `[documentation.plans].ignore` for project-relative files or directories
  that should be skipped by the plan/log audit while legacy material is being
  migrated.
- Keeps configured plan roots strict even if an ignore entry points at or
  contains a plan root.
- Documents the migration setting in the plan/log workflow guide.
